Multipurpose Zone Near Buriganga

Made A 3D design and prepared a Project Proposal for CE302 course

About This Project

This project demonstrates the complete lifecycle of a design process, beginning with the foundational stages of planning and culminating in a detailed 3D visualization. The primary objective was to apply a structured, professional approach to project management and execution from start to finish.

The initial phase involved developing a comprehensive project inception report to define the scope, goals, and constraints. Following this, a formal project proposal was created, which detailed the methodology, timeline, and expected outcomes. The final stage brought the entire concept to life through an immersive 3D design, where Enscape was used to create photorealistic renderings and virtual walkthroughs of the final model. Using Sketchup for detailed 3D modeling and Enscape for photorealistic rendering, I developed a complete conceptual design. This included planning layouts for green spaces, walkways, seating areas, and recreational facilities to maximize usability and visual appeal.
